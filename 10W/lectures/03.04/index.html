Some functions you should know how to write that pertain to arrays.

If these stump you, read 5.1 through 5.7, and see if that helps to shed some light on the situation.

  1. How to find the min or max of an array.
    1. e.g. int maxValue(int a[], int n); or int minValue(int a[], int n);
    2. Also be able to do this for an array of doubles
    3. Also, be able to do this for an array of structs, e.g. struct Point, where what you are finding is the maximum distance of any point from a given point, or from the origin.
  2. Finding the index of the maximu element in an array
    1. e.g. int indexOfMax(int a[], int n);
    2. When there are ties, be able to return either the highest such index, or the lowest such index.)
    3. Same for minimum value
    4. Be able to do this for doubles, and also for an array of structs
  3. Finding the count of all values in an array that meet some criterion,
    1. How many numbers in the array are negative? int countNegative(int a[], int n);
    2. Be able to do this for even, odd, negative, positive, divisible by k, etc.
    3. Should also be able to do this for an array of structs—for example, how many points are at distance more than d from the origin, or from some given point?
  4. Given an empty array (with its capacity and occupancy), an a full array (with its capacity and occupancy), move all, or only some, of the values from the source array into the destination array.
    1. Moving all the values is easy---just use a for loop
    2. Moving just some of the values that meet a criterion—e.g. all the odd values, or all the even values—requires a for loop with an if test.
  5. Given an array with some values (and the occupancy and the capacity), remove all the values that meet some criteria (scooting the others down).
  6. Given an array, sort the array, ascending or descending using selection sort.
  7. Given an array, accumuate a sum of all the values in that array
    1. also all values that meet some criterion
    2. also for an array of structs—sum some particular elements of the struct based on some criterion, perhaps of another member in the struct.
  8. Given an array, search for a particular value in that array, and return the index of the first occurence, or -1 if it is not found.

 

 

Something about the * operator

The following Ch session illustrates that the () are needed with the * operator in C when accessing a member of a struct.

/cs/faculty/pconrad> struct S {int a;};
/cs/faculty/pconrad> struct S x;
/cs/faculty/pconrad> x.a = 5;
5 
/cs/faculty/pconrad> struct S *p;
/cs/faculty/pconrad> p=&x;
0x8e79e90 
/cs/faculty/pconrad> p->a=7;
7 
/cs/faculty/pconrad> x.a
7 
/cs/faculty/pconrad> (*p).a=3;
3 
/cs/faculty/pconrad> x.a
3 
/cs/faculty/pconrad> *p.a = 4;
