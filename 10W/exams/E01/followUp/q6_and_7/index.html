  1. (5 pts) Very briefly: What is the purpose of using int main(int argc, char *argv[]) instead of int main() in a C program?


    Answer: This allows the user to enter input to the program on the command line, i.e. command line arguments.

  1. In programs using int main(int argc, char *argv[]) we often see lines of code such as the following:

    width = atoi(argv[1]);
    height = atoi(argv[2]);


    Before we can use these lines of code, there are three things we must do.

    1. (3 pts) There is a line of code we need before the main program to be sure we can access atoi.

      What is that line of code?

      ANSWER: #include <stdlib.h>


    2. (3 pts) We need either one, or two lines of code we need right at the start of the main program to declare width and height.

      What is that code?


      ANSWER: int width, height;

      OR:

      int width;
      int height;
