In lab02, you worked with Turtle graphics, creating letters for your initials. In this first few problems on this exam, you'll work with some code that does something similar. We will discuss these function definitions:

  • drawZero(t,w,h) Draw a "number 0" symbol (as a simple rectangle)
  • drawOne(t,w,h) Draw a "number 1" symbol.
  • drawLessThan(t,w,h) Draw the "less than" symbol.
  • drawGreaterThan(t,w,h) Draw the "greater than" symbol.
  • drawSpace(t,w) Doesn't really draw anything; just moves over to right by distance w

Turtle Graphics Reference

import turtle
load the turtle module
aTurtle = turtle.Turtle()
create a Turtle name 'aTurtle'
x = aTurtle.xcor()
set x to the x-coordinate of aTurtle's current location
y = aTurtle.ycor()
set y to the y-coordinate of aTurtle's current location
aTurtle.up()
put aTurtle's pen in the up position;
aTurtle doesn't draw lines
aTurtle.down()
put aTurtle's pen in the down position;
aTurtle will draw lines
aTurtle.forward(dist)
move aTurtle 'dist' in the direction it's facing
aTurtle.backward(dist)
move aTurtle 'dist' in the direction opposite to where it's facing
aTurtle.right(deg)
turn aTurtle 'deg' degrees to the right
aTurtle.left(deg)
turn aTurtle 'deg' degrees to the left
aTurtle.goto(x, y)
move aTurtle to the coordinate (x, y)
